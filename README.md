# Controls

| Key                  | Action     |
| -------------------- | ---------- |
| Up                   | Move Up    |
| Down                 | Move Down  |
| Left                 | Move Left  |
| Right                | Move Right |
| Enter                | Restart    |
| Swipe (Touch Screen) | Move       |
